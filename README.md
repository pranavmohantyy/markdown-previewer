# Markdown Previewer

Convert Markdown text to styled HTML in real-time.

## Features
- **Bold Text** with double asterisks: `**bold**`
- *Italic Text* with single asterisks: `*italic*`
- Headers:
  - `# Header 1`
  - `## Header 2`
  - `### Header 3`

## Usage
Run the previewer with your markdown file:
```
python preview.py yourfile.md --open --watch
```

## Example
```markdown
# My Title
This is **bold** text and this is *italic* text.
## Subtitle
### Sub-subtitle
```
